In order to use the widgets here, you must use the custom widget option in StreamElements. For information on how to do that, please visit https://dev.streamelements.com/docs/widgets

Open the Editor in the Custom widget UI list. 

Steps:

	1. Copy and paste all applicable code into the corresponding tabs of the editor.
	2. Click Done in the editor
	3. Click Save on the Overlay Setting
	4. Test
	5. Done
